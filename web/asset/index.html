How It Works

Rugburn is injected into the ijl15.dll file that ships with PangYa.

Instead of patching PangYa itself, it mostly relies on in-memory patching of Windows functions to augment PangYa's behavior, allowing it to work across different PangYa versions and regions.

Please note that Rugburn does still need to have patches to fully disable GameGuard for each region and version. In addition, Rugburn does not emulate GameGuard and can not be used for cheating on official servers.

Patch / Update

Please either:

  • Click Browse and select ijl15.dll from your PangYa install directory.
  • OR, drag and drop ijl15.dll from your PangYa install directory to the zone below.

If you select an already-patched ijl15.dll, it will be updated with the latest Rugburn patch.

Your browser will download a new ijl15.dll. Copy it over the old ijl15.dll in your PangYa install directory.

Note: This patcher tool does not upload anything to the internet. It uses only your browser.

Protip: Back up your old ijl15.dll file. That way, it's easy to undo this later.

Rugburn - Patch for PangYa™

Install Rugburn into your PangYa installation. It can:

  • Disable GameGuard
  • Rewrite Network Requests
  • Bypass the Updater, so you can run ProjectG directly.

Rugburn allows otherwise-unmodified official PangYa clients to connect to unofficial servers.

It is also open source.

To get started, patch your PangYa installation quickly and easily using the patcher tool.
